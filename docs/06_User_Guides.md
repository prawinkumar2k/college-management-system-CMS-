# User Process Guides

Detailed instructions for common user tasks within the SF ERP system.

## 1. Login and First-Time Access
1.  Navigate to `http://localhost:3000`.
2.  Select your **Role** (Admin, Staff, or Student).
3.  Enter your **Username** (Staff ID or Register Number).
4.  Enter your **Password**.
5.  Click **Sign In**.

## 2. Creating a New User (Admin Only)
1.  Go to **User Creation** in the sidebar.
2.  Select the **Staff Member** from the dropdown (this list is populated from `staff_master`).
3.  Select the **User Role**.
4.  Optionally create a new Role using the `+` button.
5.  Set a **Username** and **Password**.
6.  Select the **Access Modules** this user can access.
7.  Click **Create User**.

## 3. Recording Student Attendance
1.  Navigate to **Student Attendance Entry**.
2.  Select the **Date**, **Department**, and **Semester**.
3.  The system will load the list of students.
4.  Mark students as **Present**, **Absent**, or **OD**.
5.  Submit the entry.

## 4. Managing Memos
1.  Go to **Memo Center**.
2.  Choose between **Staff Memos** or **Student Memos**.
3.  **View**: Recent memos are listed in chronological order.
4.  **Create**: Use the "Create Memo" button to broadcast a subject and message.
5.  **Target**: Staff memos can be targeted to specific departments.

## 5. Master Data Setup
- **Staff Master**: Add new faculty members, upload photos, and set designations.
- **Semester Master**: Manage academic years and semesters.
- **Subject Master**: Define course subjects and link them to departments.

## 6. Accessing Reports
- **Dashboard**: High-level statistical report.
- **Attendance Report**: View detailed attendance history for specific students or departments.
- **Academic Reports**: View schedules and performance data (if applicable).
