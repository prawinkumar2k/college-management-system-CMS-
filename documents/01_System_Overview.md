# System Overview: SF ERP

SF ERP is a comprehensive Enterprise Resource Planning (ERP) solution designed specifically for educational institutions. It provides a centralized platform to manage various school and college operations, from student admissions and attendance to staff management, payroll, and academic planning.

## Core Purpose

The system aims to digitize and streamline administrative processes, providing real-time data access for administrators, faculty, and students. By automating routine tasks, it reduces manual errors and improves institutional efficiency.

## Key Features

- **Dashboard**: Real-time analytics and statistics (Total Students, Staff, Attendance trends).
- **User Management**: Role-based access control (Admin, Staff, Student, HOD, etc.).
- **Staff Management**: Detailed staff records, designations, and department assignments.
- **Student Management**: Registration, profile management, and academic history.
- **Attendance Tracking**: Daily attendance entry for students and staff.
- **Academic Planning**: Semester management, subject allocation, and academic calendars.
- **Communication (Memos)**: Internal memo system for broadcasting information to staff and students.
- **Caller/Enquiry Management**: Tracking prospective student enquiries and call assignments.

## Design Philosophy

- **User-Centric**: A clean, modern interface (Gradients, Glassmorphism) for ease of use.
- **Scalable**: Built on a robust Node.js and MySQL stack, deployable via Docker for consistency.
- **Secure**: Implements JWT-based authentication and BCrypt password encryption.
- **Responsive**: Accessible across desktops and tablets.
